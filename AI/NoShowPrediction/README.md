\# Dentix No-Show Prediction



This module predicts whether a patient is likely to attend or miss an appointment.



\## Files



\- no\_show\_model.py: model training / prediction script

\- no\_show\_model.pkl: trained machine learning model

\- dentix\_no\_show\_mock\_data\_7k.csv: generated dataset used for training/testing

\- dentix\_no\_show\_mock\_data.csv: sample dataset



\## Purpose



The model helps the clinic identify high-risk no-show appointments and improve appointment management.
