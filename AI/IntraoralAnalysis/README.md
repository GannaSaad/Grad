\# Dentix Intraoral Image Analysis



This module contains the notebooks used for training and evaluating the intraoral disease classification models.



\## Implemented Models



\- MobileNet

\- ResNet18

\- ResNet18 + Random Forest

\- Support Vector Machine (SVM)



\## Target Classes



\- Calculus

\- Caries

\- Gingivitis

\- Hypodontia

\- Mouth Ulcer

\- Tooth Discoloration



The corresponding dataset is located in:



Datasets/IntraoralDataset
