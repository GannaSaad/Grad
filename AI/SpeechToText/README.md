\# Dentix Speech-to-Text Module



This module provides voice documentation capabilities for Dentix.



\## Components



\### speech\_to\_text.py

Handles speech transcription using Whisper.



\### extract\_record.py

Extracts structured medical information from transcripts.



\### voice\_record\_api.py

Provides API endpoints for voice recording and transcription.



\## Features



\- Real-time speech transcription.

\- Automatic extraction of clinical records.

\- Structured medical output.

\- Supports English and Arabic speech.
